Color theory

- Light colors (shine): First color circle by Sir Isaac Newton (1666): RGB (primary colors red, green, blue), an additive color model.
- Print colors (cover) by Moses Harris (1766): RYB (primary colors red, yellow, blue as pigment of paint). Now CMY or CMYK, a subtractive color model.

Secondary colors are 2 mixed primary colors.
Tertiary colors are derived from one primary and one secondary color.
Inverse colors come from the diametrically opposite position in the color wheel. See opponent color. In CSS: 

			.class-name {
	filter: invert(100%);
}

HLS and HSV colors are alternative representations of the RGB color model. The HSL model attempts to resemble more perceptual color models such as the Natural Color System (NCS) or Munsell color system.
HLS is often used to create pastel colors (high value, low saturation)
